# Taurus Consultora

## Descripción
Desarrollo integral de una página web corporativa tipo Single Page Application. El proyecto cuenta con un diseño interactivo estructurado en múltiples secciones (Nosotros, Servicios, Proyectos, Contacto). Implementa funcionalidades como tarjetas de proyectos clickeables, contadores animados de estadísticas y un carrusel bidireccional de clientes.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
